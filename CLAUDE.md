# Diretrizes Gerais para Agentes (IA ou Desenvolvedores)

- Sempre ler docs/PRD.md antes de iniciar qualquer tarefa.
- Implementar UMA task por vez (sem “pular” tasks).
- Não fazer refatorações não relacionadas.
- Ao concluir uma task:
  - Atualizar docs/PRD.md marcando a checkbox correspondente.
  - Fazer commit pequeno e objetivo.
- Uma task só pode ser considerada concluída após confirmação explícita do usuário.
- Se houver dúvida, a task deve permanecer como pendente.
